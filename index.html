Pusat Panduan Portal Hibah

Pelajari cara menggunakan sistem Grants Portal sesuai dengan peran Anda.

add_home

A. Membuat Program Baru

Langkah bagi **Superadmin/Admin** untuk merancang skema hibah baru di dalam sistem.

1Klik tombol Tambah Program di Dashboard utama.
2Isi Nama Skema, pilih Ikon, dan Deskripsi.
3Di Dashboard Program, atur Periode (Open/Close).
4Atur Syarat Berkas dan Atribut Data di menu Pengaturan.
description

B. Mengajukan Proposal

Panduan bagi **Pengusul** untuk mengirimkan usulan proposal mereka ke sistem.

1Buka dashboard program hibah yang dituju.
2Klik tombol Usulan Baru.
3Isi judul, data atribut kustom, dan upload lampiran wajib.
4Pastikan semua mandatory terisi lalu klik Kirim Proposal.
person_add

C. Assign Reviewer

Proses penugasan evaluator oleh **Admin/Manager** ke usulan yang masuk.

1Buka tab Usulan di dashboard program tertentu.
2Klik ikon menu di samping kanan nama pengusul.
3Pilih Assign Reviewer pada baris data usulan.
4Centang daftar Reviewer yang diinginkan lalu Simpan.
rate_review

D. Evaluasi (Review)

Langkah bagi **Reviewer** untuk memberikan skor dan rekomendasi akhir.

1Buka daftar usulan yang ditugaskan kepada Anda.
2Klik usulan untuk melihat detail dan download berkas.
3Buka tab Penilaian lalu isi skor per-indikator.
4Simpan draft atau kirim penilaian final ke Admin.
